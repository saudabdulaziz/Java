Description of the lab project

1.    An imaginary (or complex) number is of the form a + bi where variables, a and b, are real numbers; i stands for the square root of -1. An example is 3.5+4.5i. The real component is 3.5; the imaginary component is 4.5. In this project, we'll need to add and multiply complex numbers.

2.    To add two complex numbers we simply add real-to-real and imaginary-to-imaginary. Consider the imaginary numbers 1.5+2.5i and 3.5+4.5i. When we add the real portions 1.5+3.5, we get 5. When we add the imaginary portions 2.5+4.5, we get 7. The answer is therefore 5+7i.

3.    To subtract two complex numbers we subtract real-to-real and imaginary-to-imaginary portions. For example 3+4i – (1+2i) = 3-1 + 4i-2i = 2 + 2i

4.    Multiplication is somewhat trickier. We describe the process of multiplying 1+2i by 3+4i in the next three steps:

a. Compute the real part of the product. First multiply real-by-real (1*3=3) and imaginary-by-imaginary (2*4=8). Then subtract the product of the imaginary parts (8) from the product of the real parts (3) to get the answer (3-8=-5). The -5 is the real part of the product.

b. Compute the imaginary part of the product. Multiply the first number's real part (1) by the second number's imaginary (4) for a result of 4. Then multiply the second number's real (3) by the first number's imaginary (2) for a result of (6). Add the results of these products together (4+6) to get 10. The 10 is the imaginary part of the product.

c. We combine the results of steps a and b to get the answer: -5+10i

5.    We can divide by converting the problem into a multiplication. First we need to define the term: a complex number conjugate. The conjugate of a complex number is formed by reversing the sign of the imaginary part. The conjugate of 1+2i is 1-2i. The trick to division is to multiply both numerator and denominator by the conjugate of the denominator. The complex terms of the denominator will cancel leaving a real number in the denominator.  Then divide both the real and imaginary parts of the conjugated numerator by this real number.

For example, to divide 1+2i by 3+4i, first multiply the numerator and denominator by the conjugate of the denominator (3-4i). This converts the division to a multiplication and the 'i' terms in the denominator cancel out. (1+2i)/(3+4i) = (1+2i)*(3-4i)/((3+4i)*(3-4i)) = (11+2i)/25.  Then divide both the real and imaginary portions of 11+2i by 25 yielding 11/25 + 2/25i or 0.44+0.08i. Use the multiplication procedure in the above step to verify this result.

// Program to add and multiply complex numbers.

// Lab #1 for cs258; uses a class that handles complex numbers (Complex.java).

 

import java.io.*;

 

public class Calculator

{  public static void main(String[] args)

   {  BufferedReader in = new BufferedReader(new InputStreamReader(System.in));

 

      Complex num1, num2, result = new Complex();

      char    op;

      boolean more;

 

      System.out.println("\nComplex number calculation program\n");

      do

      {  num1 = readComplex(in);

         num2 = readComplex(in);

 

         op   = readChar(in, "Enter the operation '+, -, *, or /': ", "+-*/");

         switch (op)

         {  case '+':  result = num1.add(num2);

                       break;

            case '*':  result = num1.product(num2);

                       break;

            case '/':  result = num1.div(num2);

                       break;

            case '-':  result = num1.minus(num2);                      

         }

         System.out.println(num1+" "+op+" "+num2+" = "+result);

         more = (readChar(in, "Perform another calculation? (y/n): ", "yn") !='n');

                 

      } while (more);

           

      System.out.println("\n\nCalculation program completed\n");

   }

     

   // Method to read a complex number

   public static Complex readComplex(BufferedReader in)

   {  

     String line;

      while (true)
      {     System.out.print("Enter a complex number: ");  
            try
            {  line = in.readLine();
               return Complex.parseComplex (line); 
            }
            catch (Exception exception)
            {  System.out.println("illegal Format - enter 'a+bi' and try again");  }
      }

   }

     

   // Method to read the first character of a string

   public static char readChar(BufferedReader in, String prompt, String chars)

   {

      String line;

      char   value;
     
      while (true)
      {  try
         {  System.out.print(prompt);
            line = in.readLine();
            value = line.toLowerCase().charAt(0);
            if (chars.indexOf(value)<0) throw new NumberFormatException();
            return value;
         }
         catch (Exception exception)
         { System.out.println("Illegal - try again"); }
      }
   }
